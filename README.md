# This is a test for the BINHS Research Archive
A browser for completed research papers of the Senior High School students of BINHS

## Administrator File Formatting Guide

Administrators must follow this format when uploading research files.

### File Requirements
- Each research entry must have **two files**:
  - **`.pdf`** – Contains the full research paper.
  - **`.txt`** – Contains the **metadata** of the research paper.

### File Purpose
- **PDF File**
  - Stores the complete research document.

- **TXT File**
  - Stores the metadata used by the system for **searching, filtering, and indexing**.

---

## TXT Metadata Format

Follow this exact format inside the `.txt` file:
-Title:
-Authors:
-Date Published:
-Grade:
-Strand:
-Section:
-Research Type:
-Format and Design:
-Subject Area:
-School Year:
-Adviser:
-Keywords:
-Award:
-Abstract:
