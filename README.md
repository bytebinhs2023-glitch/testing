# This is a test for the BINHS Research Archive
A browser for completed research papers of the Senior High School students of BINHS

## Administrator File Formatting Guide

Administrators must follow this format when uploading research files to the archive.

### File Requirements
Each research entry must include **two files**:

- **`.pdf`** – Contains the complete research paper.
- **`.txt`** – Contains the metadata of the research paper.

### File Purpose
**PDF File**
- Stores the full research document.

**TXT File**
- Stores the metadata used by the system for **searching, filtering, and indexing**.

---

## TXT Metadata Format

Follow this **exact format** inside the `.txt` file:

```
Title:
Authors:
Date Published:
Grade:
Strand:
Section:
Research Type:
Format and Design:
Subject Area:
School Year:
Adviser:
Keywords:
Award:
Abstract:
```

### Notes
- Do **not change the field names**.
- Leave a space after the colon before adding the information.
- The **Abstract** may contain multiple sentences.
- Keep all metadata fields in the **same order**.
- All research papers must have both the **PDF file** and **TXT metadata file**.
